What counts as a refund request?
This queue includes bookings where the client requested a refund/cancel action and the system logged [REFUND_REQUEST] in the booking’s admin note.
Recommended workflow
  1. Open the booking in the Bookings console.
  2. Review service context, notes, and conversation.
  3. If appropriate, force-cancel pending/accepted bookings with an admin note.
  4. Leave a clear reason for auditability.
