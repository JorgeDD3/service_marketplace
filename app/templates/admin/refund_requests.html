What counts as a refund request?
A booking should appear in this queue when the client requests a refund/cancel action and the system records it in the booking’s admin note (ex: [REFUND_REQUEST]).
Recommended workflow
  1. Open the booking in the Bookings console.
  2. Review service context, client/provider notes, and conversation.
  3. If appropriate, force-cancel pending/accepted bookings with an admin note.
  4. Leave a clear reason in the admin note for auditability.
